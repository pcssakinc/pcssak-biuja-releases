# MPL-2.0 구성요소 원본 소스 / MPL-2.0 Component Source

PCssak Biuja 0.1.0은 아래의 수정하지 않은 MPL-2.0 구성요소를 바이너리 형태로 포함합니다.
각 링크는 이번 릴리스에 고정된 정확한 버전의 원본 소스 압축 파일을 가리킵니다. 이 문서는
PCssak Biuja 자체 소스코드의 라이선스를 변경하지 않습니다.

동일한 5개 원본 패키지를 묶고 `Cargo.lock` 체크섬으로 검증한 보관본은
[`PCssak-Biuja-0.1.0-MPL-Sources.zip`](https://github.com/pcssakinc/pcssak-biuja-releases/releases/download/v0.1.0/PCssak-Biuja-0.1.0-MPL-Sources.zip)에서 받을 수 있습니다.

| 구성요소 | 버전 | 원본 소스 |
| --- | ---: | --- |
| cssparser | 0.36.0 | [crates.io 원본 압축 파일](https://crates.io/api/v1/crates/cssparser/0.36.0/download) |
| cssparser-macros | 0.6.1 | [crates.io 원본 압축 파일](https://crates.io/api/v1/crates/cssparser-macros/0.6.1/download) |
| dtoa-short | 0.3.5 | [crates.io 원본 압축 파일](https://crates.io/api/v1/crates/dtoa-short/0.3.5/download) |
| option-ext | 0.2.0 | [crates.io 원본 압축 파일](https://crates.io/api/v1/crates/option-ext/0.2.0/download) |
| selectors | 0.36.1 | [crates.io 원본 압축 파일](https://crates.io/api/v1/crates/selectors/0.36.1/download) |

전체 제3자 구성요소, 저작권 및 라이선스 전문은 저장소 루트의
[`THIRD-PARTY-NOTICES.txt`](../THIRD-PARTY-NOTICES.txt)를 확인하세요. 설치 프로그램도 같은
고지 파일을 설치 폴더에 배치합니다. 소스 링크가 작동하지 않거나 특정 구성요소의 정확한
소스 사본이 필요하면 `pcssakinc@gmail.com`으로 제품 버전과 구성요소 이름을 보내주세요.
